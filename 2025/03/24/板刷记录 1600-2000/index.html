Problem - 1264B - Codeforces

题意 给出 a, b, c, d, 分别是 0, 1, 2, 3 的数量. 用这些数字组一个序列, 使得 \(|a_i - a_{i - 1}| = 1\)

思路 注意到先排列 0 和 2, 得到: __ 0 __ 0 … 0 __ 2 __ 2 … 2 __ . 然后安排 1 和 3: 数字 1 可以插到任意间隔中, 但是 3 只能插到 2 的后面. 所以先安排 3, 然后剩下的位置填充 1. 最后检查一下即可.

Problem - 1256C - Codeforces

题意 给出 m 个长度为 \(c_i\) 的木板和长度为 n 的河. 每次最多能跳到 now + d 处. 构造一个方案, 要求使用所有木板.

Problem - 1254B1 - Codeforces

题意 给一个 n 的数组 a (\(a_i\in \{0, 1\}\)), 每次操作可以 \(a_i -1\) (不能出现负数), 然后令 \(a_{i + 1} + 1\) 或者 \(a_{i -1} + 1\) . 问: 最少操作多少次, 使得: \(\exist k\leq 2\and \forall i, k\mid a_i\).

思路 注意到 \(a_i\) 要么是 0, 要么是 1, 记录和为 s. 所以可以从 2 到 s 枚举 \(k\), 又注意到假如 \(x = p\times q\), 那么 \(k = p\) 和 \(k = q\) 一定不比 \(k = x\) 更劣, 故而只枚举 \(s\) 的质因子即可. 关于如何判断 "这一组 1 最少要几次操作". 等价于问题 \(\min \sum_l^r|x_i - p|\), 答案是 p 为 中间的点 \(x_{mid}\) ( \(mid = {l + r\over 2}\), 向上向下取整均可 ) 时最优, 即最优解是 \(\sum |x_i - x_{mid}|\), 可以 \(O(n)\) 求出. 故而复杂度 \(O(n\cdot\omega(n))\).

Problem - 1254A - Codeforces

题意 给一个 \(r\times c\) 的矩阵 a ( \(a[i][j]\in \{0, 1\}\) ) 和 k. 要求把矩阵划分成 k 个连通块, 使得所有连通块内的 1 的数量尽可能平均 . 给出一种方案.

思路 注意到 \(a[i][j] = 0\) 的地方随便染色. 故而考虑 1 的地方即可. 注意到对于相邻行, 正序倒序交替染色, 能保证一定联通. 模拟即可.

代码 Submission #318905958 - Codeforces

Problem - 1253D - Codeforces

题意 定义好图：如果结点 l 和 r 联通 ( \(l < r\) ), 那么 l 和 i 一定联通 ( \(l < i < r\) ). 给出一张点为 1 到 n 的无向图, 问: 至少要加多少条边, 才能变成好图.

思路 注意到: 结点 l 和 r 联通 ( \(l < r\) ), 那么 \([l, r]\) 是一个连通块. 故而记录每个连通块内的 r, 遍历一次即可.

代码 Submission #318910206 - Codeforces

Problem - 1252H - Codeforces

题意 给 n 个不相邻的矩形土地, 在土地上修 2 座占地为相同矩形 (可以旋转) 的大楼. 问单个大楼的最大面积

思路 当两座楼修到同一块土地时, 一定挑最大的土地. 否则的话, 问题转化成: 从 n 个矩形中挑 2 个, 使得旋转后重叠面积最大. 离散化所有矩形的两条边, 然后处理一个最值后缀. 扫一遍矩形, 每次二分找 "比当前矩形的长还长的边中, 另一条边的最值" 即可. 注意会出现二分到自己的边的情况, 所以要记录出现次数判断合法性.

Problem - 1252C - Codeforces

题意 两个长 n 的序列 r, c. 定义四联通的二维矩阵 \(a[i][j] = r[i] + c[i]\), 给出矩阵的两个点 bg 和 ed, 问: 从 bg 到 ed 是否有全偶数的边

思路 注意到奇偶加法的规律, 发现连通块都是矩形. 所以假如可以的话, 那么先固定 x 移动 y, 然后固定 y 移动 x. 记录 r 和 c 每个点右侧第一个不同奇偶的位置即可.

Problem - 1250J - Codeforces

题意 给一个数组 a 和 k, 高度为 i 的士兵有 \(a_i\) 个. 现在要挑一些组成 k 列, 要求每列的人数相同, 并且两两之间身高差值不能超过 1. 问: 最多能挑出多少人来?

思路 每列人数对于可行性显然有可二分性. 二分答案即可.

Problem - 1250B - Codeforces

题意 给一个数组, 第 i 个人属于 \(a_i\) 组 ( \(n\leq 10^5, k\leq 8\times 10^3\) ). 要求雇一个载量为 s 的车送所有队伍, 每次只能送一个队或者两个队. 记一共送了 r 次, 代价为 s * r. 问代价最小为多少?

思路 经典元素是集合编号的问题, 容易在复杂度上被忽悠. 用桶 cnt 来统计每个组有多少人, 升序排列 .考虑枚举 s, 易知 s 的下界是 \(cnt_k\). 上界呢? 考虑到当 s 固定时, 一定要尽量同时每次载两队, 所以要贪心的尽可能同时载剩下的最小组和最大组. 故而上界是 max(cnt[i], cnt[k - i + 1]).

Problem - 1237C2 - Codeforces

题意 在三维空间中给 n 个点 ( \(n\leq 5\times 10^4\) ), 每次选一对消掉, 要保证每对形成的长方体没有其他点. 给出一个方案.

思路 考虑二维的情况, 不难类比出三维的情况. 开一个二维桶, 记录 (x, y) 位置里有几个点, 两两消掉. 再开一个一维桶, 记录 x 位置上有几个点, 两两消掉. 剩下的按 x 升序两两消掉即可.
